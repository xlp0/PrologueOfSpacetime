# The Telescope (The Astronomer's Eye)

**Role**: To observe the Firefall (65,536 Probes).
**Function**: Magnification and Light Gathering.

## Hardware Required
To make this a *working* telescope, you need:

1.  **Objective Lens**: **50mm Diameter** / **360mm Focal Length**.
2.  **Eyepiece**: **1.25"** (31.75mm).
3.  **Bolts (Structure)**:
    *   **6x M3 Bolts** (12mm) + Nuts (To connect tube sections).
    *   **1x 1/4-20 Hex Nut** (For Tripod Mount).
4.  **Tube Material**: 3D Printed Sections (Bolt together).

## 3D Printing Instructions
**File**: `telescope_assembly.scad` (Robust V2)
**Parts**:
1.  **Front Tube**: Holds Lens + Front Flange.
2.  **Back Tube**: Holds Focuser + Tripod Block + Back Flange.
3.  **Lens Ring**: Retains the Objective.
4.  **Focuser**: Slides inside.

**Settings**:
*   **Infill**: 40% (Needs stiffness for optics).
*   **Supports**: **Required** (For Flanges and Tripod Block).
